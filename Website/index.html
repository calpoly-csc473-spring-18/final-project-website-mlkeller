Soft Shadows

By Mitchell Keller

Project Description

This ray tracer features soft shadows. For soft shadows, the lights can't be points because each object hit needs to sample a large amount of points on the light in order to create this soft shadow effect. Each point uses N samples and casts rays to those samples to determine if the light is obstructed or not. The radius of the light also effects how large the shadow will extend to.

Usage

Use the flag "-N=" followed by the number of samples desired.

Use the flag "-radius=" followed by the radius for all lights desired.
